# avifconv

## CLI

`Basic`

```sh
avifconv ./path_to_dir

avifconv ./path_to_file
```

`Show version`

```sh
avifconv --version
```

`Options`

```sh
# quality
avifconv --quality 0-100

# alpha channel quality
avifconv --quality-alpha 0-100

# speed
avifconv --speed 6

# threads
avifconv --workers 4
```

## Build

```sh
./build.sh
```
